Smart Cart: An Intelligent Shopping Experience

Introducing 'Smart Cart', a robust C program that revolutionizes the way we handle our shopping needs. This project embodies a seamless integration of practicality, efficiency, and thoughtful programming to provide an intelligent shopping experience.

About the Project:

  • Data Structures at Play: The project involves the use of structured data types such as 'GroceryItem', 'Bag', and 'Cart', representing a product, a shopping bag, and a shopping cart respectively. This structural organization allows us to simulate a real-life shopping experience accurately.
  • Intelligent Shopping: An intelligent algorithm is designed to pack items into the shopping bags considering constraints like maximum weight and item limit per bag. The algorithm efficiently checks the item count and weight, ensuring no bag exceeds its maximum capacity.
  • Efficient Resource Management: Using dynamic memory allocation, the project ensures optimal utilization of system resources. It dynamically allocates and deallocates memory for shopping bags based on the number of items and their weight, thus avoiding waste of memory.
  • Enhanced User Experience: Functions like 'ProductDisplay', 'BagDisplay', and 'CartDisplay' provide a user-friendly interface that displays information about the products, the bags, and the shopping cart in a clear and understandable way.
  • Safe Shopping: The 'deletePerish' function ensures perishable items are removed from the cart, contributing to a safer shopping experience. This feature showcases the application's potential in facilitating intelligent and safe shopping experiences.

Developed with an emphasis on practicality and usability, 'Smart Cart' offers a glimpse into the future of shopping. It exemplifies the power of C programming in creating efficient and functional applications that can potentially streamline our everyday tasks.
